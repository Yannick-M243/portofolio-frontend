## Installation (Windows)

-   Clone the repo
    ```sh
    git clone https://github.com/Yannick-M243/portofolio-frontend
    ```
-   Switch to the repo folder
    ```sh
    cd portofolio-frontend
    ```
-   Install all the dependencies
    ```sh
    npm install
    ```
